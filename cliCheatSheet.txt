Start a new project
ionic start projectname template(eg: sidemenu) --v2 --ts

Run locally
ionic serve

Upload to cloud
<create apps.ionic.io account>
ionic upload
<login if not already logged in>


Install app on device
<plug-in device that has developer mode enabled>
ionic run android (or ios)

Generate page
ionic g page MyPageName

config.xml
Data like app title, e-mail address etc

CHange app icon / splash screen
1. change resources/icon.png and /splash.png to the one you want
2. ionic resources --icon --splash
